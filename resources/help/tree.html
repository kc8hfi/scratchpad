The TREE Part:

You can add a node here. The node represents a subject, a title, or something
to indicate a topic for information.

  • The Tree area consists of a tree structure.  You can add nodes, delete nodes, and change the names of the nodes.  The nodes can also be moved to different locations.  
  • The Article Area is where you type in.  This is the "notepad" area;  this is where the notes are stored for the selected treenode.
