Rearranging Nodes in the Tree

To cut/copy a node:

  1. Right-click the node
  2. left-click on either the cut or copy option from the menu

You can then pick a node, then right-click and bring up the menu again, and choose "Paste" or "Paste as Sibling". See Paste and/or Paste as sibling for more information.

Paste as a Child Node

If after you use the cut or copy option, and you select the "Paste" option, the currently selected node will be the parent of the node. For example, if the recently cut node was called "stuff" and the "paste" option was selected, stuff would appear under the "article". "Article" would now become the parent of "stuff".

Paste as a Sibling Node

If after you use the cut or copy option, and you select the "Paste as Sibling" option, the node will appear right under the currently selected node. For example, if the recently cut node was called "stuff" and the "paste as Sibling" option was selected, stuff would appear under the "article". "Article" would now become a sibling of "stuff".
