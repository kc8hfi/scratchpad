Moves Selected  Node Up



  1. Select a node to be moved
  2. Click the "Move Node Up" toolbar icon
  3. The node will move up in the tree
There are 2 other ways to get to this command:
  • click on the tools menu
  • right-click on the node
There are some basic rules to this particular command.  The node will only move up if the parent is the same as the selected node.  If you want to move the node out to another parent, or to another child, use the cut and paste features.  See "cut and paste" for information.



Move Selected Node Down


  1. Select a node to be moved down
  2. Click the "Move Node Down" toolbar icon. 
  3. The node will move down in the tree
There are 2 other ways to get to this command:
  • click on the tools menu
  • right-click on the node
As with the "Move selected Node Up" option, there are some rules to this command.  The node will only move the node down if the parent node stays the same as the selected node.  If you want the node to be moved to a different location, see "cut and paste" for more information.  


Cut and Paste Nodes



To cut a node out of the tree:
  1. Right-click the node
  2. left-click the "cut" option from the menu
You can then pick a node, then right-click and bring up the menu again, and choose "Paste as Child" or "Paste as Sibling".  See "Paste As Child" for more information.  See also "Paste as sibling".  


Paste as a Child Node



If after you use the cut option,  and you select the "Paste as Child" option, the currently selected node will be the parent of the node.  For example, if the recently cut node was called "stuff" and the "paste as child" option was selected, stuff would appear under the "article".  "Article" would now become the parent of "stuff".
See also "cut and paste nodes" and "Paste as sibling".  


Paste Node as a Sibling



If after you use the cut option,  and you select the "Paste as Sibling" option, the node will appear right under the currently selected node.  For example, if the recently cut node was called "stuff" and the "paste as Sibling" option was selected, stuff would appear under the "article".  "Article" would now become a sibling of  "stuff".
See also "cut and paste nodes" and "Paste as sibling".
